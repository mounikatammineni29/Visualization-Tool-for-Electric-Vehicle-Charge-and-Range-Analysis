# ⚡ Visualization Tool for Electric Vehicle Charge and Range Analysis

An interactive **Tableau Public** dashboard that analyzes Electric Vehicle (EV) performance, charging characteristics, and vehicle specifications. The dashboard transforms raw EV data into meaningful visualizations, helping users compare different electric vehicles and gain insights into the growing EV industry.

---

## 📌 Project Overview

This project presents an interactive Tableau dashboard that enables users to:

- Analyze Electric Vehicle performance.
- Compare driving range across EV models.
- Explore battery capacity and charging information.
- Compare EV manufacturers.
- Visualize key metrics through interactive charts and filters.

---

## 🎯 Objectives

- Analyze Electric Vehicle datasets.
- Compare EV brands and models.
- Visualize battery capacity and driving range.
- Present interactive business intelligence dashboards.
- Support data-driven decision making using Tableau.
