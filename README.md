# 🚗 Electric Vehicle Analysis Dashboard

## 📌 Project Overview

The Electric Vehicle (EV) Analysis Dashboard is an interactive Tableau project designed to analyze electric vehicle adoption trends and provide insights into EV distribution, manufacturers, vehicle types, and model performance. The dashboard helps users understand the growth of electric vehicles through effective visualizations and data-driven analysis.

---

## 🎯 Objectives

- Analyze Electric Vehicle adoption trends.
- Identify the top EV manufacturers.
- Compare Battery Electric Vehicles (BEV) and Plug-in Hybrid Electric Vehicles (PHEV).
- Visualize EV distribution across different regions.
- Understand vehicle model popularity.
- Build an interactive dashboard for easy data exploration.

## 🚀 How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/govardhanreddy18/EV.git
```

2. Open the project folder.

3. Open **Book1.twb** using Tableau Desktop.

4. If prompted, reconnect the data source to:

```
ElectricCarData_Clean.csv
```

5. Explore the interactive dashboard.
